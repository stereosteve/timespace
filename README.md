# TimeSpace

Position DOM elements temporally.

    npm install
    node app.js

## Example Timelines

  - Years: Life of Oppenheimer - or - the development of the atomic bomb
  - Months: Presidential elections - separate timeline for each candidate
  - Days: Apollo 13
  - Hours: Football game
